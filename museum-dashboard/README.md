# museum-dashboard

Frontend projekt (React + Vite) pre dashboard časť múzejného systému.

## Spustenie

```bash
cd museum-dashboard
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Poznámka

Produkčný dashboard používaný Raspberry Pi backendom je servovaný z `raspberry_pi/Web/`.
Tento priečinok môže slúžiť na samostatný vývoj UI.
